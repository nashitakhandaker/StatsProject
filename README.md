## Project Description
Predicting housing prices is important for making informed real estate decisions. This analysis provides both insight into key pricing factors and a way to predict future home values, which can be useful for buyers, sellers, and real estate professionals. 

In this project, we use the Ames Housing dataset to conduct two analyses. Firstly, we examine how living area relates to sale prices across three specific neighborhoods. Secondly, we build regression models using multiple variables to predict housing prices across all of Ames.
